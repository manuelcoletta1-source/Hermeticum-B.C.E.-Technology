# Pipeline (alto livello)

1) Create (gateway) → genera IPR + output minimi  
2) Verify (offline) → hash/firma/manifest  
3) Comply → stato conformità (auditabile)  
4) Gate → decisione ex-ante (ALLOW/DENY)  
5) Exchange → scambio controllato (allow-by-proof)  
6) Registry → pubblicazione sperimentale (hash-only)
