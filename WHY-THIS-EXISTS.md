# Perché questo sistema esiste

Internet è pieno di sistemi che chiedono fiducia.

Questo non la chiede.

## Qui puoi verificare

Puoi controllare l’integrità.  
Puoi modificare il contenuto.  
Puoi provare a romperlo.  

Se il sistema non rileva una modifica → è rotto.  
Se rileva ogni modifica → è coerente.

## Perché è diverso

Non si basa su:
- autorità centrale
- certificazioni dichiarative
- fiducia nell’emittente

Si basa su:
- verifica matematica
- coerenza nel tempo
- osservabilità pubblica

## Cosa stai guardando

Un ambiente tecnico che dimostra se stesso.

Non un sito.  
Non una demo.  
Un sistema verificabile.
